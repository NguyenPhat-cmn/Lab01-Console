Các khái niệm chính

CI - Continuous Integration

Tự động build và test mỗi khi push code. Phát hiện lỗi sớm, giảm conflict khi merge.

CD - Continuous Deployment

Tự động deploy sau khi CI pass. Release nhanh, giảm thủ công, feedback sớm.

Runner

Server/container thực thi các jobs. GitLab.com cung cấp shared runners miễn phí.

Artifacts

File output được tạo ra bởi jobs, có thể chia sẻ giữa các stages trong pipeline.
